# Select placeholder

```css
&::-webkit-input-placeholder, // chrom, safari, opera
&::-moz-placeholder, // firefox 19
&:-ms-input-placeholder, // IE 10
&:-moz-placeholder,
&::placeholder {
	color: $color-gray-dark-2;
}
```
