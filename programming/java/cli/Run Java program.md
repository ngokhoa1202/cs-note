# Generate .class file
```Shell title='Command to compile Java source file into class file'
javac -d  <path-to-generate> <.java file>
```
- `-d` is directory flag.
- Example: `javac -d ./obj Main.java`
# Run .class file
- 
```Shell title='Command to run Java class file'
java -cp <path-to-class-file> <.class file without .class extension>
```
- `-cp` is class path flag.
