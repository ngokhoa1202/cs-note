# Install NodeJs
- Download `nvm` package manager from NodeJs default website.
- Employ `nvm` to install NodeJs.
# Specify NodeJs version
- Specify NodeJs version using command.
```bash title='Command to configure NodeJs version'
nvm use <version>
```
