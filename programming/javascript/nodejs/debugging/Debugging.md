# Debugging in Node.js
- Insert keyword `debugger` into the lines to be tracked.
- Run the `node inspect` command.
```Shell title='Command to debug'
node inspect <app.js>
```

- Open Google Chrome Dev tool, select the Node.js icon. The dev tool  will open a window for debugging.
