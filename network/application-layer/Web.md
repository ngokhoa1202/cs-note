# Concepts
- Known as ==WWW== <-> World Wide Web or Web 3.
- Based on ==HTTP== [HTTP](HTTP.md) protocol.
- consists of ==addressable objects== (using ==URL==):
	- file ==index.html==
	- image files.
	- stylesheet file.
	- scripts (Javascript files).
	- web APIs.
