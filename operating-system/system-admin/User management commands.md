# useradd
- Create a new user.
```bash
useradd <username>
```

# usermod
- Modify a user.

# groupadd
- Add a new group.

# chgrp
- Change the group ownership of a particular user.
```bash
chgrp -R <group> <user>
```
- `-R` flag: Recursively change on files and related directories.
