# Purpose
- Exploit `curl` to make HTTP Requests by command line.
# Syntax
```Shell
curl [options | urls]
```
## Options
## Specify HTTP Method
- `-X` or `--request` 
```Shell
curl -X GET www.google.com
```
## Show response headers
- `-i` or `--show-headers` 
## Specify username and password for authentication
- `-u` or `--user` `<username>:<password>`
